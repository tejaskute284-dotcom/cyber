# Cyber Hygiene App

A comprehensive cybersecurity awareness application designed to help users improve their digital hygiene.

## Features

-   **Dashboard**: Track your cyber score and pending security tasks.
-   **Knowledge Hub**: Interactive educational resources (Articles, Videos, Courses).
-   **Gamification**: Leaderboard and badges (CyberNinja, SecureSam).
-   **Daily Tips**: Bite-sized cybersecurity advice fetched dynamically.
-   **Chatbot**: AI-powered assistant for security questions.
-   **Secure Authentication**: User registration and login with persistent storage.

## Tech Stack

-   **Frontend**: React, TypeScript, Tailwind CSS, Framer Motion
-   **Backend**: Node.js, Express
-   **Database**: JSON-based persistent file storage (Serverless-style)
-   **Theme**: Black/Violet/Blue Glassmorphism

## Getting Started

1.  **Install Dependencies**
    ```bash
    npm install
    cd server && npm install
    ```

2.  **Run the App**
    ```bash
    # Run Client (Root)
    npm run dev

    # Run Server (in /server)
    cd server && npm run dev
    ```

3.  **Access**
    Go to `http://localhost:5173`
